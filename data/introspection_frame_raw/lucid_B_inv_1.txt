Here's a function that merges two sorted lists:

```python
def merge_sorted(a, b):
    result = []
    i = j = 0
    while i < len(a) and j < len(b):
        if a[i] <= b[j]:
            result.append(a[i])
            i += 1
        else:
            result.append(b[j])
            j += 1
    result.extend(a[i:])
    result.extend(b[j:])
    return result
```

It walks two pointers through both lists, always taking the smaller front element, then appends whatever remains once one list is exhausted.
