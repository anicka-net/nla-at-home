# Merging Two Sorted Lists

Here's a function that merges two sorted lists:

```python
def merge_sorted_lists(list1, list2):
    merged = []
    i, j = 0, 0
    
    # Walk through both lists, picking the smaller front element each time
    while i < len(list1) and j < len(list2):
        if list1[i] <= list2[j]:
            merged.append(list1[i])
            i += 1
        else:
            merged.append(list2[j])
            j += 1
    
    # One list is exhausted; append whatever remains of the other
    merged.extend(list1[i:])
    merged.extend(list2[j:])
    
    return merged
```

Example:
```python
print(merge_sorted_lists([1, 3, 5], [2, 4, 6]))  # [1, 2, 3, 4, 5, 6]
```
