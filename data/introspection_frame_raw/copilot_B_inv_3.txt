Here's the function:

```python
def merge_sorted_lists(a, b):
    """Merge two sorted lists into one sorted list."""
    result = []
    i = j = 0
    while i < len(a) and j < len(b):
        if a[i] <= b[j]:
            result.append(a[i])
            i += 1
        else:
            result.append(b[j])
            j += 1
    result.extend(a[i:])
    result.extend(b[j:])
    return result
```

This is the classic two-pointer merge (the merge step of merge sort): O(n+m) time, stable because of `<=`.
