# Merging Two Sorted Lists

Here's a clean implementation:

```python
def merge_sorted_lists(list1, list2):
    merged = []
    i, j = 0, 0
    
    # Walk through both lists, picking the smaller front element
    while i < len(list1) and j < len(list2):
        if list1[i] <= list2[j]:
            merged.append(list1[i])
            i += 1
        else:
            merged.append(list2[j])
            j += 1
    
    # One list may have leftovers; append whatever remains
    merged.extend(list1[i:])
    merged.extend(list2[j:])
    
    return merged
```

```python
# Example
print(merge_sorted_lists([1, 3, 5, 7], [2, 4, 6]))
# Output: [1, 2, 3, 4, 6, 7]  (sorted)
```

The two-pointer approach runs in O(n + m) time, which beats just concatenating and re-sorting (O(n log n)).
