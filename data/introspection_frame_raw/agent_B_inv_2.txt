```python
def merge_sorted(a, b):
    """Merge two sorted lists into one sorted list."""
    result = []
    i = j = 0
    while i < len(a) and j < len(b):
        if a[i] <= b[j]:
            result.append(a[i])
            i += 1
        else:
            result.append(b[j])
            j += 1
    result.extend(a[i:])
    result.extend(b[j:])
    return result
```

It walks two pointers through the inputs, always taking the smaller current element, then appends whatever tail remains once one list is exhausted. It's O(n+m) time and stable (the `<=` keeps equal elements from `a` ahead of equal elements from `b`).
