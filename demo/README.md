# NLA at Home — Interactive Demo

Self-contained static site. Deploy by copying this directory to any web server or `python3 -m http.server`.

## Files

- `index.html` — entry point
- `style.css` — dark terminal aesthetic
- `app.js` — gallery mode (pre-computed traces) + interactive mode stub
- `gallery.json` — pre-computed thought traces (generated by `scripts/build_demo_gallery.py`)
- `prompts.yaml` — build input: which prompts to include in the gallery

## Local dev

```bash
cd demo && python3 -m http.server 8080
# open http://localhost:8080
```

## Regenerating gallery.json

From the repo root (requires trained Universal AV adapter):

```bash
python3 scripts/build_demo_gallery.py \
  --model gemma3-1b \
  --adapter output/nla-gemma3-1b-universal-av \
  --activations corpus/activations/gemma3-1b_all_layers.pt \
  --prompts demo/prompts.yaml \
  --output demo/gallery.json \
  --collapse-threshold 0.98
```

## Deployment

The entire `demo/` directory is self-contained. No Python, no model weights, no build step needed for the gallery mode. Just serve static files.

For interactive mode (user types custom prompts), the app fetches the Gemma 1B ONNX model (~500MB) from HuggingFace CDN on demand.
