How does this work?

A Natural Language Autoencoder (NLA) translates neural network activations into human-readable descriptions. We trained one on Phi-4 Mini (3.8B parameters, 32 layers) using contrastive GRPO to ensure descriptions carry real activation-specific information.

Each row shows what the model's internal state "looks like" at that layer depth. Early layers handle tokenization and syntax. Middle layers form concepts and detect intent. Late layers prepare the output format and tone.

The descriptions were generated by injecting the actual activation vector into the NLA at a special token position — the NLA never sees the original prompt text.

Built with NLA at Home — open replication of Anthropic's NLA. Model: microsoft/Phi-4-mini-instruct.
